## Взаимодействие с матрицой

### Создание
```
from matrix import Matrix

m = Matrix([
    [1, 9, 5, 4],
    [4, 2, 3, 7],
    [7, 2, 4, 9],
    [8, 1, 4, 2]
])
```

## Решение Линейных Алгебраических Уровнений

```
from matrix import solve_with_Kramer, solve_with_matrix_transformation, solve_with_Haus_method
```

### Матрица коэфицентов
```
A = 
╒═══╤════╤════╕
│ 2 │ -1 │  0 │
├───┼────┼────┤
│ 1 │  2 │ -1 │
├───┼────┼────┤
│ 0 │  1 │  1 │
╘═══╧════╧════╛
```
### Матрица правой части
```
b = 
╒═══╕
│ 1 │
├───┤
│ 2 │
├───┤
│ 2 │
╘═══╛
```

x = solve_with_Haus_method(A, b)
